## 金山翻译
### **功能:**
* 中英文翻译
* 音标
* 发音(***默认美式***)
* 复制(当前选中行文本,内容分为:音标,释意几行展示,可以自由复制)
    * 注:复制的同时都会有读音
        * 复制示例:`am:[stɑr] ,en:[stɑ:(r)]`

### 按键
*dc*
> 对应中文的单词,而且也离左手近,比较方便(主要是懒)
